# PyPicture
![suteru](https://github.com/user-attachments/assets/d047812e-4e02-4a20-997f-7ae1b7a828da)
pythonを使用した画像編集アプリです。
画像編集の肝にはopencvをGUIにはPySide6を使用しています。
# 機能
1. トリミング
2. 画像回転
3. 明るさ変更
4. コントラスト変更
5. シャドウ調整
6. 彩度変更
7. 色調整(RGB)
8. テキスト入力
9. ステッカー(画像差し込み)
10. レタッチ(顔面自動モザイク)

# 対応拡張子
bmp, dib, pbm, pgm, ppm, pnm, pxm, pfm, sr, png, JPEG, WebP, AVIF, tiff
